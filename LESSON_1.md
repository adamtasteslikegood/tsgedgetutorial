# Lesson 1: The Foundation

Welcome to the first lesson of our Advanced Mode tutorial! Today, we lay the groundwork for building our AI teaching assistant. We will be using a modern stack tailored for flexibility, scale, and performance.

## Core Concepts

### 1. TypeScript
TypeScript is a strongly typed programming language that builds on JavaScript, giving you better tooling at any scale. We use it to ensure our chat logic and server infrastructure are robust and less prone to runtime errors.

### 2. Node.js
Node.js is a runtime environment that allows us to run JavaScript (and by extension, compiled TypeScript) on the server side, outside of a web browser. Our console chat and Express API will run on Node.

### 3. The Interactive Console
Before building a complex Web UI with Angular, we will master the interaction loop using the terminal. The terminal (or console) gives us a straightforward environment to test our AI chat logic, history tracking, and streaming responses without the overhead of the browser.
